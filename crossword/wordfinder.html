Word Finder



The word finder will enumerate a list of words which satisfy the criteria you provide. Use the text box above to enter a term, and use the dropdown to specify what words you wish to generate given this prompt. You may specify multiple criteria by using the '+' button to add more inputs. Examples of what sort of words the different criteria generate is given here:
Means Like
Words which have a related meaning
Sounds Like
Words which are pronounced similarly
Spelled Like
Words which are spelled similarly. You can use the special characters * (which matches any number of characters) and ? (which matches exactly one character) in this mode
Associated nouns
Nouns which are commonly modified by the given adjective, e.g. gradual → increase
Associated adjectives
Adjectives which are commonly modified by the given noun, e.g. beach → sandy
Synonyms
Thesaurus style synonyms, e.g. ocean → sea
Triggers
Words which are commonly associated, e.g. cow → milking
Antonyms
Thesaurus style antonyms, e.g. late → early
Hypernyms
Generalisations, e.g. gondola → boat
Hyponyms
Specialisations, e.g. boat → gondola
Holonyms
Things which comprise the word, e.g. car → accelerator
Meronyms
Things which the word is part of, e.g. trunk → tree
Frequent Followers
Words which often come afterwards, e.g. wreak → havoc
Frequent Predecessors
Words which often come before, e.g. havoc → wreak
Rhymes
Perfect rhymes, e.g. spade → aid
Approximate Rhymes
Non-perfect rhymes, e.g. forest → chorus
Homophones
Words which are pronounced identically, e.g. course → coarse
Consonant Match
Words which differ only by vowels, e.g. sample → simple
